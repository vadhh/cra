# Legal Doc Verifier (LDV) - Pilot

Automated Contract Risk Assessment platform using LightML analysis pipeline and LLM commentary generator.

## Structure
* `ldv-backend`: Flask rest APIs and ML processing logic.
* `ldv-frontend`: HTML & TailwindCSS responsive dashboard.
* `datasets`: Rules and citation databases.
